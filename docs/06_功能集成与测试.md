# 06 — 功能集成与测试

## 6.1 整体数据流

```
┌──────────┐    frameReady     ┌──────────────┐    QImage     ┌──────────┐
│ V4l2Camera│ ──────────────→  │  MainWindow   │ ───────────→ │  LCD屏幕  │
│  (线程)   │    QImage        │  processImage │  paintEvent  │  显示     │
└──────────┘                   │  ↓            │              └──────────┘
                               │ ImageProcessor│
                               │  CLAHE/去畸变  │
                               │  去雾/锐化/降噪│
                               └──────────────┘
                                      │
                                      ↓ (如果录像中)
                               ┌──────────────┐
                               │ VideoWriter   │
                               │  保存AVI文件   │
                               └──────────────┘
```

## 6.4 功能测试用例

### 基础功能

| 测试项 | 操作 | 预期结果 |
|--------|------|----------|
| 启动 | 运行程序 | 屏幕显示摄像头画面，状态栏显示"运行中" |
| 拍照 | 点击拍照按钮或按 C 键 | ./captures/ 下生成 JPG 文件，状态栏提示文件名 |
| 录像开始 | 点击录像按钮或按 R 键 | 按钮文字变为"停止录像"，状态栏显示"录像中" |
| 录像停止 | 再次点击录像按钮 | ./videos/ 下生成 AVI 文件，按钮恢复 |
| 冻结 | 点击冻结按钮或按空格 | 画面停止更新，按钮变为"继续" |
| 解冻 | 再次点击 | 画面恢复实时更新 |
| 旋转 | 点击旋转按钮 | 画面依次旋转 90°/180°/270°/0° |
| 退出 | 按 ESC 键 | 程序正常退出，无崩溃 |

### 图像处理

| 测试项 | 操作 | 预期结果 |
|--------|------|----------|
| CLAHE | 勾选 CLAHE 复选框 | 画面暗区细节增强，整体对比度提升 |
| 畸变校正 | 勾选畸变校正 | 画面边缘的桶形畸变消除（需标定文件） |
| 去雾 | 勾选去雾 | 模糊/发白的画面变清晰 |
| 锐化 | 勾选锐化 | 边缘更清晰，纹理更明显 |
| 降噪 | 勾选降噪 | 噪点减少，画面更平滑 |
| 组合 | 同时勾选 CLAHE + 畸变校正 | 两种效果叠加，帧率 ≥15fps |

### 异常测试

| 测试项 | 操作 | 预期结果 |
|--------|------|----------|
| 无摄像头 | 不接 OV5640 启动程序 | 状态栏显示"摄像头打开失败"，不崩溃 |
| 拔摄像头 | 运行中拔掉摄像头 | 状态栏显示错误信息，不崩溃 |
| 无标定文件 | 删除 camera_calib.yml | 畸变校正复选框可勾选但无效果，不崩溃 |
| 磁盘满 | 磁盘空间不足时拍照 | 提示保存失败，不崩溃 |
| 长时间运行 | 连续运行1小时 | 无内存泄漏，帧率稳定 |

## 6.5 效果对比截图

建议在测试过程中保存对比截图，用于最终演示：

```bash
# 在开发板上创建截图目录
mkdir -p ./demo_screenshots

# 保存各算法效果对比
# 1. 原始画面
# 2. CLAHE 增强后
# 3. 去畸变后
# 4. 去雾后
# 5. 锐化后
# 6. 降噪后
```

可以在程序中加一个"保存对比图"的功能：

```cpp
void MainWindow::saveComparison() {
    QImage original = m_currentRawImage;  // 原始帧
    QImage processed = processImage(original);  // 处理后

    // 拼接左右对比图
    QImage comparison(original.width() * 2, original.height(),
                      QImage::Format_RGB888);
    QPainter p(&comparison);
    p.drawImage(0, 0, original);
    p.drawImage(original.width(), 0, processed);
    p.drawText(10, 20, "原始");
    p.drawText(original.width() + 10, 20, "处理后");
    p.end();

    comparison.save("./demo_screenshots/comparison.jpg");
}
```
