## Step 1.5：PWM 调光（补光灯亮度控制）

### 为什么需要 PWM

你现在的 LED 只有亮/灭两种状态。内窥镜的补光灯需要**连续调节亮度**（0~100%），PWM 是唯一方案。

### CubeMX 配置

```
TIM3 → Channel 1 → PWM Generation CH1
  ├─ Prescaler (PSC): 63        → 64MHz / 64 = 1MHz
  ├─ Counter Period (ARR): 999  → 1MHz / 1000 = 1kHz PWM 频率
  └─ Pulse (CCR): 0             → 初始占空比 0%（LED 灭）

引脚：选择一个支持 TIM3_CH1 的引脚（如 PC6）
```

### 代码实现

**tim.c 中添加 TIM3 初始化（CubeMX 自动生成）**

**app_control.h 中添加接口：**

```c
void LED_SetBrightness(uint8_t percent);
uint8_t LED_GetBrightness(void);
```

**app_control.c 中添加实现：**

```c
static uint8_t current_brightness = 0;

void LED_SetBrightness(uint8_t percent) {
    if (percent > 100) percent = 100;
    current_brightness = percent;
    uint32_t pulse = (uint32_t)percent * 10;  // ARR=999, 100%=1000
    __HAL_TIM_SET_COMPARE(&htim3, TIM_CHANNEL_1, pulse);
}

uint8_t LED_GetBrightness(void) {
    return current_brightness;
}
```

**App_Init() 中启动 PWM：**

```c
void App_Init(void) {
    GPIO_FixLedOutput();
    HAL_TIM_Base_Start_IT(&htim2);       // 定时器中断
    HAL_TIM_PWM_Start(&htim3, TIM_CHANNEL_1);  // PWM 输出
    HAL_UART_Receive_IT(&huart3, &rx_byte, 1);
    // ...
}
```

## Step 1.6：ADC 温度采集

### 为什么需要 ADC

内窥镜法规要求镜头前端温度不超过 41°C。M4 需要实时监控温度，超限时自动关闭补光灯。

### 硬件连接（NTC 热敏电阻）

```
3.3V ─── [10K 固定电阻] ─┬─── ADC 引脚（如 PA4）
                          │
                     [NTC 热敏电阻]
                          │
                         GND
```

### CubeMX 配置

```
ADC1 → IN4 (PA4)
  ├─ Resolution: 12 bits
  ├─ Sampling Time: 239.5 Cycles（慢速采样，精度高）
  └─ Continuous Conversion: Disabled（软件触发单次采样）
```

### 代码实现

**app_control.h 中添加：**

```c
float App_ReadTemperature(void);
void App_SafetyCheck(void);
```

**app_control.c 中添加：**

```c
#include <math.h>
#include "adc.h"

#define TEMP_LIMIT  41.0f   // 医疗法规温度上限

float App_ReadTemperature(void) {
    HAL_ADC_Start(&hadc1);
    HAL_ADC_PollForConversion(&hadc1, 10);
    uint32_t adc_val = HAL_ADC_GetValue(&hadc1);

    // ADC 值 → 电压
    float voltage = adc_val * 3.3f / 4096.0f;

    // 电压 → NTC 电阻值（分压公式）
    float resistance = 10000.0f * voltage / (3.3f - voltage);

    // 电阻值 → 温度（Steinhart-Hart 简化公式）
    // B 值 = 3950，参考温度 25°C = 298.15K，参考电阻 10K
    float temp = 1.0f / (logf(resistance / 10000.0f) / 3950.0f
                 + 1.0f / 298.15f) - 273.15f;
    return temp;
}

// 在 TIM2 中断中调用（每 100ms 检查一次）
void App_SafetyCheck(void) {
    static uint8_t overheat_cnt = 0;
    float temp = App_ReadTemperature();

    if (temp > TEMP_LIMIT) {
        overheat_cnt++;
        if (overheat_cnt >= 3) {  // 连续 300ms 超温才触发
            LED_SetBrightness(0);
            LED_SetAll(GPIO_PIN_SET);
            UART_SendString("[SAFETY] OVERHEAT! LED OFF\r\n");
            overheat_cnt = 3;  // 防止溢出
        }
    } else {
        overheat_cnt = 0;
    }
}
```

**App_TimerCallback() 中添加安全检查：**

void App_TimerCallback(void) {

    // 安全检查（每 100ms）
