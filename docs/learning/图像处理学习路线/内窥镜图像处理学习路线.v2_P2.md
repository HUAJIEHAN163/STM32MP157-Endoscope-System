**为什么重要：** 现代内窥镜产品的核心竞争力是 AI 辅助诊断（息肉检测、病变分类）。

**嵌入式端模型部署流程：**

```
PC 端训练                    嵌入式端部署
┌──────────┐                ┌──────────────────┐
│ PyTorch  │  → 导出 ONNX → │ ONNX Runtime     │
│ 训练模型  │                │ 或 TFLite        │
│          │  → 量化压缩  → │ 在 A7 上推理     │
```

**ONNX Runtime 部署（推荐方案）：**

```cpp
    Ort::Env env;

    PolypDetector(const char *model_path)

    vector<Rect> Detect(const Mat &frame) {

        Mat blob;
        resize(frame, blob, Size(416, 416));
        blob.convertTo(blob, CV_32F, 1.0 / 255.0);

        // 转为 NCHW 格式
        vector<float> input_data(3 * 416 * 416);
        // BGR → RGB，HWC → CHW
        for (int c = 0; c < 3; c++)
            for (int h = 0; h < 416; h++)
                for (int w = 0; w < 416; w++)
                    input_data[c * 416 * 416 + h * 416 + w] =
                        blob.at<Vec3f>(h, w)[2 - c];

        // 推理
        auto input_tensor = Ort::Value::CreateTensor<float>(
            Ort::MemoryInfo::CreateCpu(OrtArenaAllocator, OrtMemTypeDefault),
            input_data.data(), input_data.size(),
            vector<int64_t>{1, 3, 416, 416}.data(), 4);

        const char *input_names[] = {"input"};
        const char *output_names[] = {"output"};
        auto outputs = session.Run(Ort::RunOptions{nullptr},
                                   input_names, &input_tensor, 1,
                                   output_names, 1);

        // 解析输出为检测框
```

### 10. 图像拼接与 3D 重建

**为什么重要：** 高端内窥镜需要将多帧图像拼接成全景图，辅助医生全面观察。

**特征匹配（拼接基础）：**

```cpp
// ORB 特征匹配（速度快，适合嵌入式）
Mat StitchFrames(const Mat &frame1, const Mat &frame2) {
    // 1. 提取特征点
    Ptr<ORB> orb = ORB::create(500);
    vector<KeyPoint> kp1, kp2;
    Mat desc1, desc2;
    orb->detectAndCompute(frame1, noArray(), kp1, desc1);
    orb->detectAndCompute(frame2, noArray(), kp2, desc2);

    // 2. 匹配
    BFMatcher matcher(NORM_HAMMING);
    vector<DMatch> matches;
    matcher.match(desc1, desc2, matches);

    // 3. 筛选好的匹配
    sort(matches.begin(), matches.end());
    int good_count = matches.size() * 0.3;
    vector<DMatch> good_matches(matches.begin(),
                                matches.begin() + good_count);

    // 4. 计算变换矩阵
    vector<Point2f> pts1, pts2;
    for (auto &m : good_matches) {
        pts1.push_back(kp1[m.queryIdx].pt);
        pts2.push_back(kp2[m.trainIdx].pt);
    }
    Mat H = findHomography(pts2, pts1, RANSAC);

    // 5. 拼接
    Mat result;
    warpPerspective(frame2, result, H,
                    Size(frame1.cols * 2, frame1.rows));
    frame1.copyTo(result(Rect(0, 0, frame1.cols, frame1.rows)));

    return result;
}
```

### 11. 视频编码与存储

**为什么重要：** 内窥镜检查过程需要录像存档，用于后续诊断和医疗记录。

```cpp
// 使用 FFmpeg 进行硬件编码（A7 侧）
// 编译时链接：-lavcodec -lavformat -lavutil -lswscale

#include <opencv2/opencv.hpp>

class VideoRecorder {
    VideoWriter writer;
    bool recording = false;

public:
    void StartRecording(const string &filename, Size frame_size) {
        // H.264 编码，文件体积小
        writer.open(filename,
                    VideoWriter::fourcc('H', '2', '6', '4'),
                    30, frame_size);
        recording = true;
    }

    void WriteFrame(const Mat &frame) {
        if (recording && writer.isOpened())
            writer.write(frame);
    }

    void StopRecording() {
        recording = false;
        writer.release();
    }
};
```

### 12. 医疗图像标准与法规

**为什么重要：** 医疗器械软件必须符合法规，否则无法上市。

**必须了解的标准：**

| 标准 | 内容 | 与你的关系 |
|------|------|-----------|
| IEC 62304 | 医疗器械软件生命周期 | 代码必须有版本管理、测试记录 |
| IEC 62366 | 可用性工程 | UI 设计需要符合人因工程 |
| ISO 13485 | 质量管理体系 | 开发流程必须可追溯 |
| DICOM | 医学图像标准格式 | 图像存储和传输格式 |

**DICOM 格式基础：**

