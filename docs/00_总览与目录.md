# STM32MP157 内窥镜图像处理系统 — 开发指南

## 项目目标

在野火 STM32MP157 开发板上，使用 OV5640 摄像头 + 开发板屏幕，开发一个**单机内窥镜图像处理 Demo**。

展示能力：嵌入式 Linux 系统开发、V4L2 摄像头驱动、OpenCV 图像处理、Qt GUI 开发。

## 硬件清单

| 硬件 | 型号 |
|------|------|
| 开发板 | 野火 STM32MP157 Pro/Mini |
| 摄像头 | OV5640（MIPI CSI / DVP 接口） |
| 显示屏 | 开发板配套 LCD（通常 800x480 或 480x272） |
| 调试 | USB-TTL 串口线、网线/WiFi |

## 整体开发流程图

```
┌─────────────────────────────────────────────────────────┐
│  Phase 1: 环境准备（第1-2天）                             │
│  安装交叉编译工具链 → 交叉编译Qt/OpenCV → 烧录系统镜像      │
│  → 确认OV5640驱动加载 → v4l2-ctl验证摄像头                 │
├─────────────────────────────────────────────────────────┤
│  Phase 2: 基础功能（第3-5天）                             │
│  V4L2采集模块 → Qt显示窗口 → 实时预览跑通                   │
├─────────────────────────────────────────────────────────┤
│  Phase 3: 图像处理（第6-8天）                             │
│  CLAHE增强 → 畸变校正 → 去雾/降噪 → 白平衡                │
├─────────────────────────────────────────────────────────┤
│  Phase 4: 完善功能（第9-10天）                            │
│  拍照 → 录像 → 冻结画面 → 图像旋转 → 亮度对比度调节         │
├─────────────────────────────────────────────────────────┤
│  Phase 5: 优化与文档（第11-12天）                         │
│  性能优化 → 代码整理 → README → 演示准备                   │
└─────────────────────────────────────────────────────────┘
```

## 最终 Demo 预期效果

- 开发板上电后自动启动内窥镜程序
- LCD 屏幕实时显示 OV5640 摄像头画面
- 界面上可切换多种图像处理算法（CLAHE、去畸变、去雾、锐化）
- 支持拍照保存、视频录制
- 支持画面冻结、旋转、亮度/对比度调节
- 显示实时帧率、时间戳
