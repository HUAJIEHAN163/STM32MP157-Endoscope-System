# 05 — 内窥镜图像处理算法

## 5.1 为什么选择这些算法

内窥镜工作环境的特殊性决定了需要特定的图像处理：

| 问题 | 原因 | 对应算法 |
|------|------|----------|
| 图像暗、细节不清 | 体腔内光照不均匀，LED光源有限 | CLAHE 自适应直方图均衡化 |
| 镜头畸变 | 内窥镜使用广角/鱼眼镜头 | 镜头畸变校正 |
| 画面模糊/起雾 | 体腔内温差导致镜头起雾 | 去雾算法（暗通道先验） |
| 噪声大 | 低光环境 + 小尺寸传感器 | 双边滤波降噪 |
| 边缘不清晰 | 组织纹理细腻 | USM 锐化 |
| 色偏 | LED 光源色温不准 | 白平衡校正 |

## 5.2 算法模块设计

```cpp
// image_processor.h
#ifndef IMAGE_PROCESSOR_H
#define IMAGE_PROCESSOR_H

#include <opencv2/opencv.hpp>

class ImageProcessor {
public:
    // CLAHE 自适应直方图均衡化（内窥镜最常用）
    static void applyCLAHE(const cv::Mat &src, cv::Mat &dst,
                           double clipLimit = 3.0,
                           cv::Size tileSize = cv::Size(8, 8));

    // 镜头畸变校正
    static void initUndistortMap(const cv::Mat &cameraMatrix,
                                  const cv::Mat &distCoeffs,
                                  cv::Size imageSize,
                                  cv::Mat &map1, cv::Mat &map2);

    // 去雾（暗通道先验，简化版）
    static void applyDehaze(const cv::Mat &src, cv::Mat &dst,
                            double omega = 0.95, int radius = 7);

    // 锐化（USM）
    static void applySharpen(const cv::Mat &src, cv::Mat &dst,
                             double sigma = 3.0, double amount = 1.5);

    // 降噪（双边滤波）
    static void applyDenoise(const cv::Mat &src, cv::Mat &dst,
                             int d = 5, double sigmaColor = 50,
                             double sigmaSpace = 50);

    // 白平衡（灰度世界法）
    static void applyWhiteBalance(const cv::Mat &src, cv::Mat &dst);
};

#endif // IMAGE_PROCESSOR_H
```

## 5.3 算法实现

### image_processor.cpp

```cpp
#include "image_processor.h"

// ============================================================
// 算法1：CLAHE 自适应直方图均衡化
// ============================================================
// 原理：将图像分成小块（tile），对每块分别做直方图均衡化，
//       并用 clipLimit 限制对比度放大倍数，避免噪声被放大。
// 内窥镜意义：体腔内光照不均匀，CLAHE 能同时增强亮区和暗区的细节。
//
void ImageProcessor::applyCLAHE(const cv::Mat &src, cv::Mat &dst,
                                 double clipLimit, cv::Size tileSize) {
    cv::Mat lab;
    cv::cvtColor(src, lab, cv::COLOR_BGR2Lab);

    std::vector<cv::Mat> channels;
    cv::split(lab, channels);

    // 只对 L 通道（亮度）做 CLAHE，保留色彩信息
    cv::Ptr<cv::CLAHE> clahe = cv::createCLAHE(clipLimit, tileSize);
    clahe->apply(channels[0], channels[0]);

    cv::merge(channels, lab);
    cv::cvtColor(lab, dst, cv::COLOR_Lab2BGR);
}

// ============================================================
// 算法2：镜头畸变校正
// ============================================================
// 原理：根据相机标定得到的内参矩阵和畸变系数，
//       预计算像素映射表（remap），实时校正每一帧。
// 内窥镜意义：广角镜头产生严重桶形畸变，影响医生对组织形态的判断。
//
// 标定步骤（离线完成）：
// 1. 用内窥镜拍摄棋盘格标定板（至少15张不同角度）
// 2. 用 cv::findChessboardCorners 找角点
// 3. 用 cv::calibrateCamera 计算内参和畸变系数
// 4. 将结果保存到文件，程序启动时加载
//
void ImageProcessor::initUndistortMap(const cv::Mat &cameraMatrix,
                                       const cv::Mat &distCoeffs,
                                       cv::Size imageSize,
                                       cv::Mat &map1, cv::Mat &map2) {
    cv::initUndistortRectifyMap(cameraMatrix, distCoeffs,
                                 cv::Mat(), cameraMatrix,
                                 imageSize, CV_16SC2,
                                 map1, map2);
}

// 实时校正时只需调用：
// cv::remap(src, dst, map1, map2, cv::INTER_LINEAR);

// ============================================================
// 算法3：去雾（暗通道先验，简化版）
// ============================================================
// 原理：何恺明的暗通道先验理论——在无雾图像的非天空区域，
//       至少有一个颜色通道的值接近0。利用这个先验估计雾浓度并去除。
// 内窥镜意义：体腔内温差导致镜头起雾，画面变白变模糊。
//
void ImageProcessor::applyDehaze(const cv::Mat &src, cv::Mat &dst,
                                  double omega, int radius) {
    cv::Mat srcFloat;
    src.convertTo(srcFloat, CV_64FC3, 1.0 / 255.0);

    // 1. 计算暗通道
    cv::Mat darkChannel;
    std::vector<cv::Mat> channels;
    cv::split(srcFloat, channels);
    darkChannel = cv::min(cv::min(channels[0], channels[1]), channels[2]);

    cv::Mat kernel = cv::getStructuringElement(cv::MORPH_RECT,
                                                cv::Size(radius, radius));
    cv::erode(darkChannel, darkChannel, kernel);

    // 2. 估计大气光值（取暗通道最亮的0.1%像素对应的原图均值）
    double maxVal;
    cv::minMaxLoc(darkChannel, nullptr, &maxVal);
    cv::Scalar A = cv::mean(srcFloat, darkChannel > maxVal * 0.9);

    // 3. 估计透射率
    cv::Mat transmission;
    cv::Mat normImg = srcFloat.clone();
    for (int i = 0; i < 3; i++)
        channels[i] /= A[i];
    cv::merge(channels, normImg);

    cv::split(normImg, channels);
    transmission = cv::min(cv::min(channels[0], channels[1]), channels[2]);
    cv::erode(transmission, transmission, kernel);
    transmission = 1.0 - omega * transmission;

    // 限制最小透射率，避免除零
    cv::max(transmission, 0.1, transmission);

    // 4. 恢复无雾图像
    cv::split(srcFloat, channels);
    for (int i = 0; i < 3; i++)
        channels[i] = (channels[i] - A[i]) / transmission + A[i];
    cv::merge(channels, dst);

    dst.convertTo(dst, CV_8UC3, 255.0);
}

// ============================================================
// 算法4：USM 锐化
// ============================================================
// 原理：用高斯模糊得到低频分量，原图减去低频得到高频细节，
//       再将高频细节按比例叠加回原图。
// 内窥镜意义：增强组织纹理和边缘，帮助医生观察细微病变。
//
void ImageProcessor::applySharpen(const cv::Mat &src, cv::Mat &dst,
                                   double sigma, double amount) {
    cv::Mat blurred;
    cv::GaussianBlur(src, blurred, cv::Size(0, 0), sigma);
    cv::addWeighted(src, 1.0 + amount, blurred, -amount, 0, dst);
}

// ============================================================
// 算法5：双边滤波降噪
// ============================================================
// 原理：在空间域和像素值域同时加权滤波，
//       既能平滑噪声，又能保留边缘（不像高斯滤波会模糊边缘）。
// 内窥镜意义：低光环境噪声大，但又不能模糊组织边缘。
//
void ImageProcessor::applyDenoise(const cv::Mat &src, cv::Mat &dst,
                                   int d, double sigmaColor,
                                   double sigmaSpace) {
    cv::bilateralFilter(src, dst, d, sigmaColor, sigmaSpace);
}

// ============================================================
// 算法6：白平衡（灰度世界法）
// ============================================================
// 原理：假设场景中所有颜色的平均值应该是灰色，
//       据此计算各通道的增益系数进行校正。
// 内窥镜意义：LED 光源色温偏差导致画面偏蓝/偏黄。
//
void ImageProcessor::applyWhiteBalance(const cv::Mat &src, cv::Mat &dst) {
    std::vector<cv::Mat> channels;
    cv::split(src, channels);

    double avgB = cv::mean(channels[0])[0];
    double avgG = cv::mean(channels[1])[0];
    double avgR = cv::mean(channels[2])[0];
    double avg = (avgB + avgG + avgR) / 3.0;

    channels[0] *= (avg / avgB);
    channels[1] *= (avg / avgG);
    channels[2] *= (avg / avgR);

    cv::merge(channels, dst);
}
```

## 5.4 镜头标定（离线步骤）

畸变校正需要先标定相机。这是一个离线步骤，只需做一次。

### 标定工具代码

```cpp
// calibrate_tool.cpp — 在宿主机上运行即可
#include <opencv2/opencv.hpp>
#include <iostream>
#include <vector>

int main() {
    // 棋盘格参数
    cv::Size boardSize(9, 6);  // 内角点数
    float squareSize = 20.0f;  // 格子边长(mm)

    std::vector<std::vector<cv::Point3f>> objectPoints;
    std::vector<std::vector<cv::Point2f>> imagePoints;

    // 生成世界坐标
    std::vector<cv::Point3f> objp;
    for (int i = 0; i < boardSize.height; i++)
        for (int j = 0; j < boardSize.width; j++)
            objp.push_back(cv::Point3f(j * squareSize, i * squareSize, 0));

    // 读取标定图片（用内窥镜拍摄的棋盘格照片）
    for (int i = 1; i <= 20; i++) {
        std::string filename = "calib_" + std::to_string(i) + ".jpg";
        cv::Mat img = cv::imread(filename);
        if (img.empty()) continue;

        cv::Mat gray;
        cv::cvtColor(img, gray, cv::COLOR_BGR2GRAY);

        std::vector<cv::Point2f> corners;
        bool found = cv::findChessboardCorners(gray, boardSize, corners);
        if (found) {
            cv::cornerSubPix(gray, corners, cv::Size(11, 11),
                             cv::Size(-1, -1),
                             cv::TermCriteria(cv::TermCriteria::EPS +
                                              cv::TermCriteria::COUNT, 30, 0.1));
            imagePoints.push_back(corners);
            objectPoints.push_back(objp);
            std::cout << "找到角点: " << filename << std::endl;
        }
    }

    // 标定
    cv::Mat cameraMatrix, distCoeffs;
    std::vector<cv::Mat> rvecs, tvecs;
    double rms = cv::calibrateCamera(objectPoints, imagePoints,
                                      cv::Size(640, 480),
                                      cameraMatrix, distCoeffs,
                                      rvecs, tvecs);
    std::cout << "RMS 重投影误差: " << rms << std::endl;
    std::cout << "相机内参:\n" << cameraMatrix << std::endl;
    std::cout << "畸变系数:\n" << distCoeffs << std::endl;

    // 保存到文件
    cv::FileStorage fs("camera_calib.yml", cv::FileStorage::WRITE);
    fs << "camera_matrix" << cameraMatrix;
    fs << "dist_coeffs" << distCoeffs;
    fs.release();

    std::cout << "标定结果已保存到 camera_calib.yml" << std::endl;
    return 0;
}
```

## 5.5 各算法性能参考（STM32MP157, 640x480）

| 算法 | 预估耗时 | 说明 |
|------|----------|------|
| CLAHE | 5-10ms | 只处理 L 通道，较快 |
| 畸变校正 (remap) | 3-5ms | 预计算映射表后很快 |
| 去雾 | 20-40ms | 计算量较大，可考虑降采样处理 |
| USM 锐化 | 3-5ms | 一次高斯模糊 + 加权 |
| 双边滤波 | 15-30ms | 计算量大，d 不要设太大 |
| 白平衡 | 2-3ms | 简单的通道均值计算 |

**建议**：不要同时开启所有算法。推荐默认开启 CLAHE + 畸变校正，其他按需开启。

```cpp
void applyDehazeFast(const cv::Mat &src, cv::Mat &dst) {
```
